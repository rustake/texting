String helpers for rust

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
texting = ">=0.0.1"
```

and this to your crate root:

```rust
extern crate texting;
```

## Compatibility

The `texting` crate is tested for rustc 1.48 and greater.
